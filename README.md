# NotKeeper

NoteKeeper, React Native kullanarak geliştirilmiş basit ve şık bir not tutma uygulamasıdır. Bu uygulama, kullanıcıların not eklemesini, görüntülemesini ve silmesini sağlayan sezgisel ve modern bir arayüz sunar. Özellikler:

- Yeni not ekleyin
- Mevcut notları görüntüleyin ve düzenleyin
- Notları silin
- Şık ve kullanıcı dostu tasarım

## Özellikler
- **Modern UI**: Temiz ve estetik bir tasarım.
- **Kullanıcı Dostu**: Kolay gezinme ve etkileşim.
- **Responsive**: Çeşitli ekran boyutlarına uyumlu.

## Başlarken
1. Reposu klonlayın: `git clone https://github.com/kullanıcıadınız/NotDefterim.git`
2. Proje dizinine gidin: `cd NoteKeeper`
3. Bağımlılıkları yükleyin: `npm install` veya `yarn install`
4. Uygulamayı başlatın: `npm start` veya `yarn start`

## Kullanılan Teknolojiler
- React Native
- JavaScript
- Expo
